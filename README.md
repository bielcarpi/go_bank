# Go Bank

Bank API written in Go

### Requirements
To run the PostgreSQL database in a Docker container, run:
```bash
docker run --name postgres16 -e POSTGRES_USER=root -e POSTGRES_PASSWORD=root -p 5432:5432 -d postgres:16-alpine
```
